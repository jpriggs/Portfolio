about me

"I'm a self-taught full-stack software engineer searching for full-time opportunities to improve upon my full-stack development skills, learn mobile app creation, and support platform based designs that deliver the highest quality user experience."

In 2016, I began taking Harvard University's highly respected CS50 course. It taught me the fundamentals of computer science, such as data structures, object oriented programming, time/space complexities, algorithmic logic, as well as how to do full-stack development using Python.

Since then, I have mastered front-end and back-end development earning full-stack certifications, while becoming proficient in JavaScript, Node.js, React.js, D3.js, SQL, MongoDB, and other modern tools that create interactive, highly dynamic websites.

My goal is to work for a forward-thinking company that will challenge me in a team-based agile envirnoment that empowers me to solidify my current skills, learn new frameworks and tools, as well as be deeply integrated in design implementation from planning to distribution.
